- `lib/api.ts` only sends the JSON content-type when a body exists вЂ” don't "fix" it back;
  body-less DELETEs 400 otherwise (FST_ERR_CTP_EMPTY_JSON_BODY).

  synchronously right before a synchronous compute (see scanner.ts for the pattern).
- Scanner evaluates the last CLOSED bar (still-forming bar trimmed) вЂ” same as the alert engine.
- Zustand store is a God store pending split (rebuild S7) вЂ” add slices, don't grow it.
- React 19 RC + Next 15: HMR occasionally serves stale chunks after mid-edit saves вЂ” clear
  `.next` + restart if imports "disappear".
